Web Development
===============
-Developing a website:

1. FrontEnd : User Interface ( Look & Feel)
    HTML : Content ( paragraph , div, image, table, form, list, audio, video)
    CSS :  Styling ( border , color, widht , height, background, placement )
    Bootstrap : CSS Library ( pre-defined css classes / responsive)
    Javascript : Behaviour 
    Angular/React : Javascript frameworks ( RAD , CLI ,template system , Routing, form handling)
2. BackEnd : Develop REST APIs ( Java / .net / PHP / Python / Go / NodeJS )
3. Database : Holds the data ( oracle / mysql / mongoDB )
